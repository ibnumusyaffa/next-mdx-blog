## 1. Array.map()

`Array.map()` akan membuat array baru berdasarkan hasil dari operasi setiap element array yang ada

```javascript
let umur = [10, 15, 20, 25, 30, 35, 40];

let umur2Kali = umur.map((item) => {
  return item * 2;
});

console.log(umur2Kali);
//[20, 30, 40, 50, 60, 70, 80]
```

## 2. Array.filter()

`Array.filter()` akan membuat array baru yang berisi element array yang memenuhi kriteria saja.

```javascript
let umur = [10, 15, 20, 25, 30, 35, 40];

let umurBaru = umur.filter((item) => {
  //cari umur 25 tahun keatas
  return item >= 25;
});

console.log(umurBaru);
//[25, 30, 35, 40]
```

## 3. Array.every()

`Array.every()` akan mengembalikan nilai `true` jika **semua element** array memenuhi kriteria yang ditentukan.

```javascript
let umur = [10, 15, 20, 25, 30, 35, 40];

let diatas10tahun = umur.filter((item) => {
  //cari umur 25 tahun keatas
  return item >= 10;
});

console.log(diatas10tahun);
//true, semua element memiliki nilai >= 10
```

## 4. Array.some()

jika **salah satu** element memenuhi kriteria yang ditentukan, `Array.some()` akan mengembalikan nilai `true`, berbeda dengan `Array.every()` dimana semua element harus memenuhi kriteria yang ada.

```javascript
let umur = [10, 15, 20, 25, 30, 35, 40];

let diatas10tahun = umur.some((item) => {
  //cari umur 25 tahun keatas
  return item >= 40;
});

console.log(diatas10tahun);
//true, ada satu element yang memiliki nilai >=40
```

## 5. Array.find()

`Array.find()` akan mengembalikan satu element pada array, yang pertama kali memenuhi kriteria yang ditentukan, jika tidak ada yang memenuhi kriteria, maka akan mengembalikan nilai `undefined`

```javascript
let umur = [10, 15, 20, 25, 30, 35, 40];

let cari = umur.find((item) => {
  return item >= 23;
});

console.log(cari);
// 25

let cariLagi = umur.find((item) => {
  return item >= 50;
});

console.log(cariLagi);
// undefined
```

## 6. Array.includes()

`Array.includes()` akan menentukan apakah suatu array mempunyai value tertentu didalamnya dengan mengembalikan nilai `true` atau `false`

```javascript
const pets = ["kucing", "anjing", "merpati", "kelinci"];

console.log(pets.includes("kucing"));
// true
```

## 7. Array.findIndex()
`Array.findIndex()` akan mengembalikan nomor `index` element yang pertama memenuhi kriteria yang ditentukan

```javascript
const array1 = [5, 12, 8, 130, 44];

const isLargeNumber = (element) => element > 13;

console.log(array1.findIndex(isLargeNumber));
// 3, index ke 3 element pertama yang mempunyai nilai > 13

```
